# Running Python Dummy Data

## Prerequisites
- Python 3.8+ installed
- Paho MQTT library:

```bash
pip install paho-mqtt
```

## Run the Dummy Data
Open a terminal and go to the folder:

```bash
cd Y3S2_flood2
cd python_dummy_data
```

### Run the script for Ayer Keroh:

```bash
python ayer_keroh.py
```

### Run the script for Durian Tunggal:

```bash
python durian_tunggal.py

//Press CTRL+C to stop the script.
```
